A reStructuredText plugin filter for pyspelling.

# Installation

You can install resplendent for
[Python](https://www.python.org/) via
[pip](https://pypi.org/project/pip/)
from [PyPI](https://pypi.org/).

```
$ pip install resplendent
```




## Prerequisites:
- docutils
- pyspelling
